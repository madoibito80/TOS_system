# TOSシステム
Tsukuba Official Staffsで運用される出欠確認サイト

## セットアップ
1 . /attendance/以下をサーバのドキュメントルート直下にコピーする.例 : http://ドメイン/attendance/...  
2 . /attendance/conf.phpにデータベースとメールアドレスの設定をする.  
3 . /attendance/admin/db_setup.phpにアクセスしてデータベースの初期化をする.
